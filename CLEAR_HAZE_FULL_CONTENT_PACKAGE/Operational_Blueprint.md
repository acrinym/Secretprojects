# Clear Haze — Operational Blueprint

## Manufacturing
Choose NutraCap for small-batch pilot → scale to Captek for mass runs.

## Inventory Flow
1. Order bulk ingredients (Longvida has 4-week lead time)
2. Deliver to manufacturer
3. Pilot run (1,000 bottles)
4. Full run (16,000+ units)
5. Ship to fulfillment partner (ShipBob / 3PL)

## Fulfillment
- Shopify integration
- Subscription engine
- Email/SMS flows

## Customer Support
- Simple refund policy
- Automated education emails
- Reactivation flows

## Scaling Path
Q1: Pilot + soft launch  
Q2: Ads + influencers  
Q3: National wholesale  
Q4: Subscription dominance
